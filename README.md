# KimoBot

Bot feito para realizar diversas ações realizadas a animes e mangás.

### Comandos
#### prefixo: !
##### register
Utilizado para autenticar o bot para gerenciamento do [Anilist](https://anilist.co/)

##### info \<tipo> \<busca>
Utilizado para buscar informações de animes/mangás.

```!info anime naruto```
